How this works

Odds come from Ballpark Pal Positive EV (Positive-EV.php with UseMyBooks=0). Pick a Market to set BetMarket on that page (or All for every market). Rows use two-way devig across the books you include; Ballpark Pal’s BP column is devigged the same way and folded into the consensus fair (median with book fairs). BP is not a bettable book—the Best column only picks among sportsbooks. Optional: set MLB_SCANNER_ODDS_SCREEN=1 on the server to overlay Odds Screen prices.

Devig by limits which sportsbooks supply over/under pairs for per-book devig. Fair from narrows which books feed the consensus median (BP fair is still merged in when both O and U BP prices exist). Boost adjusts EV, implied, Kelly, and best odds only.

Fast tips: Park factors are an extra HTTP request; add ?skipPf=1 to the URL or set MLB_SCANNER_SKIP_PARK_FACTORS=1 on the server for a quicker first paint.
