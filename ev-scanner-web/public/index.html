How this works

This page loads Ballpark Pal’s Positive EV board on the server, builds the same +EV table logic as the R Shiny app (fair prob from BPP “Win %” when present, else implied odds), and renders a plain HTML table—no R, no Python, no heavy widgets.

Fast tips: Park factors are an extra HTTP request; add ?skipPf=1 to the URL or set MLB_SCANNER_SKIP_PARK_FACTORS=1 on the server for a quicker first paint.
